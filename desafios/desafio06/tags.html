Tags em HTML

Em HTML (Hyper Text Markup Language) nós podemos inserir as seguintes tags, passe o cursor do mouse sobre cada elemento para ver sua especificação;

  • Títulos
  • Parágrafos
  • Imagens
  • Abreviações
  • Código Fonte
  • Listas numeradas
  • Listas com marcadores (Não ordenadas)
  • Adicionar links
  • Colocando imagens que se adaptam à tela
  • Colocando áudios no site
  • Colocando vídeos com hospedagem própria no site
