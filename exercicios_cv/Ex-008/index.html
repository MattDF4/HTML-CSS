Itálico e/ou Ênfase

Nestra frase temos um termo em destaque utilizando a Tag <i> (Não semântico.)

Nesta frase temos um termo em destaque utilizando a Tag <em> (É semântico)

TEXTO MARCADO

Texto com uma marcação utilizando a Tag <mark>

Temos outrotexto marcado no final

TEXTOS GRANDES E PEQUENOS

Colocando o texto menor, utilizadno a Tag <small> (Geralmente utilizado para aglo como 'importante' nos textos)

Marcando textos. Texto deletado:

Marcando um texto que pode ser lido, mas não considerado utilizando a Tag <del>

Texto inserido:

Para dizer e dar ênfase que o texto foi adicionado posteriormente. Utilizadno a Tag <ins> {É semântica}

Texto sobrescrito

Para coisas do tipo: x2 + 3 (X elevado a 2), usando a Tag <sup>

Texto subscrito

Para coisas como H2O, utilizando a Tag <sub>
