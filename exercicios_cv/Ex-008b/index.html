Inserindo Códigos no texto

Quando for inserir códigos de alguma linguagem de programação no texto, utilizar as Tgas <code> e <pre>. A tag 'Code' irá padronizar o tamanho de cada letra ou número digitado, e tudo deve estar dentro da Tag 'Pre', para que ela faça a identação corretamente, segue o exemplo:

Utilizando as duas Tags:

        
    num = int(input('Insira um número: '))
    if num % 2 == 0:
        print(f'O número {num} é PAR.')
    else:
        print(f'O número {num} é ÍMPAR.')
    print('FIM DO PROGRAMA.')

Utilizadno apenas a Tag 'Code':

num = int(input('Insira um número: ')) if num % 2 == 0: print(f'O número {num} é PAR.') else: print(f'O número {num} é ÍMPAR.') print('FIM DO PROGRAMA.')
